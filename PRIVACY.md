# ScrollBrake Privacy Policy

Last updated: March 18, 2026

ScrollBrake is designed to work locally in your browser.

## What ScrollBrake Stores

ScrollBrake stores the following data in the browser’s local extension storage on your device:

- site settings selected by the user
- custom domains added by the user
- usage timer data
- lockout timer data

## What ScrollBrake Does Not Do

ScrollBrake does not:

- create user accounts
- send analytics or telemetry
- upload browsing history to a server
- sync data to a ScrollBrake backend
- sell or share personal data

## How Data Is Used

Stored data is used only to:

- remember which sites you want to track
- calculate remaining usage time
- enforce temporary lockouts after a time limit is reached
- render the popup and settings interface accurately

## Permissions

ScrollBrake requests browser permissions only for features required to make the extension work:

- `storage` to save settings and timer state locally
- `tabs` to read the active tab URL so the popup can show the relevant site status
- host access on visited pages so the extension can detect tracked sites and apply the lock screen when needed

## Data Retention And Control

All stored data remains on your device unless your browser profile syncs extension storage through a browser feature you enable separately.

You can remove ScrollBrake data at any time by:

- changing or clearing settings inside the extension
- removing the extension from your browser
- clearing browser extension storage through your browser tools
